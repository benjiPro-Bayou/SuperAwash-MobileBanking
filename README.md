# Super Awash Mobile Banking App

This is a React Native (Expo) prototype for the "Super Awash" mobile banking application.

## Features

1.  **Super App Dashboard**: Integrated view of Banking, Payments, and Third-party services.
2.  **Chat Banking**: Contextual banking actions (Send, Request, Topup) directly within the chat interface.
3.  **Transfer Hub**: Unified interface for internal, external, and wallet transfers.
4.  **Mini Apps Marketplace**: Access to Ecommerce, Ride Hailing, and Travel services.
5.  **Card Management**: Visual card controls and limit settings.

## Tech Stack

*   **Framework**: React Native (Expo)
*   **Navigation**: React Navigation (Stack + Bottom Tabs)
*   **Icons**: Lucide React Native
*   **Design**: Custom styling using Awash Bank brand colors.

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the app:
    ```bash
    npx expo start
    ```

3.  Scan the QR code with your phone (Expo Go app) or run on an emulator:
    *   Press `a` for Android Emulator.
    *   Press `i` for iOS Simulator.

## Project Structure

*   `src/components`: Reusable UI components.
*   `src/screens`: Application screens.
*   `src/navigation`: Navigation configuration.
*   `src/constants`: Theme and global constants.
